# opencv_calibration
- Download 
  ```
  git clone https://github.com/alandszhang/opencv_calibration.git
  ```
- Change the parameters of calibration board and datasets in the main.cpp
- Build
  ```
  mkdir build
  cd build
  cmake ..
  make
  sudo make install
  ```
- Run the tool
  ```
  ./calibration
  ```
